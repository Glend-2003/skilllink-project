# User Service 

This microservice handles user profile management, including creating, reading, updating, and deleting user information (CRUD). It connects to a MySQL database to persist user data.

##  Tech Stack
- **Framework:** NestJS
- **Language:** TypeScript
- **Database:** MySQL
- **ORM:** TypeORM
- **Validation:** class-validator

##  Getting Started

### 1. Prerequisites
Ensure you have the following installed:
- Node.js (LTS version)
- MySQL Server (Running)

### 2. Installation
Navigate to the project directory and install the dependencies:

# Install Node modules
```bash
npm install
```
# Run in development mode
```bash
npm run start:dev
```
