# API Gateway 

The centralized entry point for the SkillLink application. It routes incoming requests to the appropriate microservices (Auth Service, User Service, etc.) and handles initial request processing.

## 🛠 Tech Stack
- **Framework:** NestJS
- **Language:** TypeScript

## Getting Started

### 1. Prerequisites
- Node.js (LTS version)

### 2. Installation
Navigate to the project directory and install the dependencies:
# Install Node modules
```bash
npm install --save-dev nodemon
```
# Run in development mode
```bash
npm run start:dev
```
